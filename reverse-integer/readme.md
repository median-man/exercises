# Reverse Integer
Write a function which takes a 32-bit signed integer and returns a number for the digits in reverse
order. The [original problem][problem] may be found at [leetCode][lc].

**Examples:**
```
reverse(123) // 321
reverse(-123) // -321
reverse(120) // 021
```
